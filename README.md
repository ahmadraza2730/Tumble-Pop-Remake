# 🎮 Tumble Pop Remake

A C++ recreation of the classic **Tumble Pop** arcade game, built as a Programming Fundamentals project during my first year of BS Computer Science at FAST-NUCES.

## 📖 About

This project reimagines the retro arcade classic Tumble Pop using core C++ concepts — no external game engine, just raw logic, structures, and problem-solving. It was built to strengthen fundamentals in control flow, data structures, and game-loop design before moving into more advanced OOP-based game development.

## ✨ Features

- 🕹️ Classic Tumble Pop-style gameplay mechanics
- 💥 Collision detection system
- 🧱 Level and obstacle logic
- 🎯 Score tracking system
- ⚙️ Built entirely with core C++ (no game engine)

## 🛠️ Tech Stack

- **Language:** C++
- **Concepts used:** Arrays/Structures, Functions, Control Flow, Basic Game Loop Design

## 🚀 Getting Started

### Prerequisites
- A C++ compiler (e.g., g++, MinGW, or Visual Studio)

### Run Locally
```bash
git clone https://github.com/ahmadraza2730/Tumble-Pop-Remake.git
cd Tumble-Pop-Remake
g++ main.cpp -o TumblePop
./TumblePop
```
